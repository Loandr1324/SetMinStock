Правила работы:
1. Все Файлы, которые выгружаются из базы, должны сохраняться в папку "Исходные данные".
2. В наименования файлах по продажам, должно быть слово "продажи".
Выгружаем продажи только из двух баз ШАВ и ШСВ.
3. В наименованиях файлов по МО, должно быть слово "МО".
Выгружаем МО из трёх баз ШАВ, ШСВ, ТС
ШАВ и ШСВ выгружаем необходимый нам МО по одному из каждого подразделения.
ТС выгружаем один МО необходимый нам с 05 Павловский и один Компания MaCar МО внешний.
4. Чтобы не ошибиться с наименованием файлов рекомендуем пересохранять файлы из 1С в файлы из 
папки "Исходные данные" под теми же именами.
5. Запускаемый файл должен находится в той же папке, что и папка "Исходные данные".
6. Вводим количество дней за которое сформированы отчёты по продажам (обычно 365).
7. Вводим количество дней, на которое нам необходимо держать запасы (обычно 90).
6. Программа создаст файл с именем "Анализ МО по компании.xlsx"

Алгоритм расчёта МО:
1. Учитываем продажи по всем розничным магазинам. Не учитываем в расчетах продажи на складе 05 Павловский и не выгружаем их для анализа.
2. Считаем общие продажи как сумму продаж по всем розничным складам кроме продаж на 05 Павловский.
Задаём название колонки "Компания MaCar продажи"
3. Делим продажи каждого магазина на количество анализируемых дней продаж (365) и умножаем на количество дней запаса (90).
Округляем в меньшую сторону.
4. Если получается значение > 0 и < 1(т.е. были какие-то продажи), то проставляем 1.
5. При подстановке МО на складах не учитываем приоритет складов. Устанавливаем МО согласно расчётам.
Так же если на каком-то складе установлено МО дробное значение, т.е. 0 < МО < 1, то значения не меняем ни в каком случае.
Даже если при расчете получается МО = 0,5. Это связано с тем, что сотрудники могут специально поставить какое-то служебное значение,
например для нового товара.
6. Если продажи по всей компании равны 0, тогда устанавливаем на всех складах МО = 0,5 включая и 05 Павловский.
7. Если продажи по всей компании равны 1, то устанавливаем МО = 1 на складе 02 Автолюбитель. На остальных розничных складах 0,33.
8. Если продажи по всей компании равны 2
8.1. Если продажа 2шт. была на одном складе, то устанавливаем МО = 1 на только на этом складе.
8.2. Если эти продажи были на разных складах, то устанавливаем МО = 1 на складе 02 Автолюбитель и МО=1 на один из складов,
на котором была продаж по приоритету:
['01 Кирова МО расчёт', '03 Интер МО расчёт', '04 Победа МО расчёт', '08 Центр МО расчёт', '09 Вокзалка МО расчёт'],
на остальных складах устанавливаем МО = 0,33.
Если на складе по приоритету, где были продажи, значение 0 < МО < 1, то его пропускаем и ставим МО = 1 на следующем
складе, где были продажи согласно приоритету (если на нём это позволяет значение МО).
8.3. Если продажа была на Автолюбителе, то устанавливаем МО согласно продажам. Где продаж не было устанавливаем МО = 0,33.
9. Если продажи по всей компании от 2 до 10 и на каком-то складе нет продаж, то устанавливаем на этом складе 0,33
10. Если продажи по всей компании кроме 05 Павловский >= 10, то на всех складах, где расчет МО = 0 устанавливаем 1.
11. Если продажи по всей компании кроме 05 Павловский >= 25, то устанавливаем значение в колонку "05 Павловский расчет"
равное (Общая сумма продаж делённое нацело на 10).
Пример:
25 = 2
29 = 2
30 = 3
206 = 20
271 = 27
10. Если продажи по всей компании кроме 05 Павловский < 25, то устанавливаем значение в колонку 05 Павловский расчет
равное 0.
12. Суммируем целые числа со всех колонок "... МО расчёт", включая "05 Павловский расчет" и подставляем это значение в колонку "Компания MaCar МО расчёт"
13. Проверяем, если продажи по всей компании = 0, то в колонку "Компания MaCar МО расчёт" подставляем значение 0.5.
14. Если МО в колонке "Компания MaCar МО" > 0 и < 1, то устанавливаем значение этого МО.
