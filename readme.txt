Правила работы:
1. Все Файлы, которые выгружаются из базы, должны сохраняться в папку Установка МО ответственным.
2. В наименования файлах по продажам, должно быть слово "продажи".
3. В наименоваях файлов по МО, должно быть слово "МО".
4. Чтобы не ошибится с наименованием файлов рекомендуем пересохранять файлы из 1С в файлы из 
папки под теми же именами.
5. Запускаемый файл должен находится в той же папке, что и папка Установка МО ответственным.
6. Программа создаст файл с именем Анализ МО по компании.xlsx
